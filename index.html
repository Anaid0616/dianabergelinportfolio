About Me.

Hi, I'm Diana Bergelin — a Frontend Developer based in Upplands Väsby, Stockholm. With a background in hairstyling and a passion for design, I bring creativity and precision to every project I work on. I build responsive and accessible web applications using semantic HTML, modern CSS and JavaScript, and I work with frameworks like React, Next.js and TypeScript with a strong focus on UX, performance and clean structure. I’m motivated by solving problems, refining user experiences, and continuously learning new technologies. Outside of coding, I enjoy photography and working on creative projects at home.
